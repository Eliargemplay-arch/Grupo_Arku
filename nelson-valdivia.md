Comandos que utilice.


git config --global user.name.
Sirve para configurar el nombre del usuario en Git.

git config --global user.email.
Sirve para configurar el correo del usuario en Git.

git config --global --list
sirve para ver toda la configuración global de Git que tienes en tu computadora.

git init
sirve para iniciar un repositorio Git nuevo en una carpeta.

git statu
Sirve para ver el estado actual de tu repositorio.

git clone
Permite copiar un repositorio remoto en mi computadora.

git checkout main
Se utiliza para cambiar a la rama principal del proyecto.

git checkout -b 
Crea una nueva rama con mi nombre y me mueve a esa rama.

git branch
Muestra todas las ramas existentes y en cuál estoy trabajando.

git add 
Agrega mi archivo al área de preparación antes de guardarlo.

git commit -m
Guarda los cambios realizados con un mensaje descriptivo.

git push -u origin
Sube mi rama al repositorio remoto para compartir mi trabajo.
